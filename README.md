# Market Risk & Quantitative Finance Toolkit
### Author: Benjawan Prukpermpool

A comprehensive quantitative finance project integrating **market risk**, **derivatives pricing**, **portfolio optimization**, and **actuarial-finance models**.  

This repository is designed to showcase technical readiness for graduate studies in **Financial Engineering**, including mathematical modeling, numerical methods, probability theory, risk analytics, and Python implementation.

It reflects a unique background bridging **actuarial science** and **banking experience**, with applications to modern quantitative finance.

---

# 🔷 Project Modules Overview

## 1️⃣ **FRTB SA — Market Risk Standardized Approach**  
Folder: `1_frtb_sa_market_risk`

Includes:
- Sensitivity-based method implementation  
- Delta, vega, and curvature aggregation  
- Bucket/cross-bucket correlation modeling  
- Capital charge computation  
- Modular Python class for testing scenarios  

📌 *Demonstrates regulatory market risk modeling + quantitative analytics.*

---

## 2️⃣ **Derivatives Pricing**  
Folder: `2_derivatives_pricing`

Contains:
- **Black-Scholes analytic pricer** (calls/puts)  
- **Binomial Tree (CRR) model**  
- **Monte Carlo simulation under GBM**  
- Clean and modular Python scripts  

📌 *Demonstrates stochastic calculus, numerical methods, simulation, and pricing theory.*

---

## 3️⃣ **Portfolio Optimization Models**  
Folder: `3_portfolio_optimization`

Includes:
- **Markowitz Efficient Frontier** generator  
- **Minimum Variance Portfolio** (closed-form solution)  
- **Risk Parity Portfolio** via iterative algorithm  

📌 *Demonstrates optimization, covariance modeling, linear algebra, and risk budgeting.*

---

## 4️⃣ **Actuarial–Finance Models**  
Folder: `4_actuarial_finance`

Includes:
- **Survival model + financial discounting**  
- **Credit migration via Markov chain**  
- **Expected loss model (PD × LGD × EAD)**  

📌 *Demonstrates actuarial thinking applied to credit risk & finance.*
